# Claude Code Help

Welcome! Here's what you can do with Claude Code.

## Quick Commands

| Command | Description |
|---------|-------------|
| `/getting-started` | New to Claude Code? Start here |
| `/help-guide` | Get contextual help and tips |
| `/mcp` | See available MCP integrations |

## What Can Claude Code Do?

### Code
- Write new code from natural language descriptions
- Explain and document existing code
- Debug errors and fix issues
- Refactor and improve code quality

### Files
- Create, edit, and search files
- Navigate your project structure
- Find patterns across your codebase

### Terminal
- Run shell commands
- Execute scripts and tests
- Manage packages and dependencies

## Tips for Best Results

1. **Be specific** - "Create a React component that shows a user profile with avatar, name, and bio" works better than "make a component"

2. **Share context** - Paste error messages, file contents, or describe your project structure

3. **Ask follow-ups** - If something isn't clear, keep asking until it makes sense

4. **Describe outcomes** - Say what you want to achieve, not just what you want to do

## Need More Help?

Just describe what you're trying to do in plain language. Claude Code will figure out the best way to help.

Example: "I want to add a dark mode toggle to my app" - Claude Code will ask clarifying questions if needed, then help you implement it step by step.
