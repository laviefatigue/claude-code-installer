# Starting Claude Code

## First Time Setup

The first time you run `claude`, you'll need to sign in:

1. Type `claude` in the terminal and press Enter
2. A browser window opens
3. Sign in with your **Claude Pro**, **Max**, or **Teams** account
4. Come back to VS Code — you're connected!

## What You'll See

```
╭─────────────────────────────────────────╮
│                                         │
│   Welcome to Claude Code!               │
│                                         │
│   Type a message to start chatting.     │
│                                         │
╰─────────────────────────────────────────╯

You:
```

Now you can type anything — ask a question, describe what you want to build, or paste an error message.

## Example

```
You: Create a simple HTML page that says "Hello World" with nice styling
```

Claude will write the code, explain it, and create the file for you.

## Tips

- Be specific: "Add a blue header" is better than "make it look nice"
- Share context: "I'm new to coding" helps Claude adjust its explanations
- Ask follow-ups: "Can you add a button?" continues the conversation
