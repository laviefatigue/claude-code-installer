# Your First Conversation

## How to Talk to Claude

You can:

**Ask questions**
```
What does this error mean?
How do I center a div in CSS?
Explain what async/await does
```

**Describe what you want**
```
Create a todo list app
Make a script that renames my photos by date
Build a simple calculator
```

**Get help with existing code**
```
Why isn't this working?
Make this code faster
Add error handling to this function
```

## What Claude Can Do

- **Write code** in any language
- **Create files** and organize projects
- **Run commands** like tests or builds
- **Edit existing code** based on your instructions
- **Explain** anything you don't understand
- **Debug errors** and suggest fixes

## A Real Example

**You:** Create a webpage with a button that counts how many times it's clicked

**Claude:** I'll create a simple click counter for you...

*[Claude writes the HTML, CSS, and JavaScript, saves the file, and explains each part]*

**You:** Make the button bigger and change the color to blue

**Claude:** I'll update the styles...

*[Claude edits just the CSS, keeping your code intact]*

## It's a Conversation

You don't need to get everything right in one message. Start simple, see what Claude creates, then refine it together.
