# Built-in Commands

Commands start with `/` and give Claude special instructions.

## Essential Commands

| Command | What it does |
|---------|--------------|
| `/help` | Shows available commands and tips |
| `/getting-started` | Interactive tutorial for new users |
| `/commit` | Saves your work with a message |
| `/clear` | Starts a fresh conversation |

## How to Use Them

In a Claude conversation, just type the command:

```
You: /help
```

Claude will respond with guidance.

## Project Commands

Some commands help with specific tasks:

| Command | What it does |
|---------|--------------|
| `/review` | Reviews code for issues |
| `/test` | Runs your tests |
| `/fix` | Attempts to fix errors |

## Creating Your Own

As you get more comfortable, you can create custom commands for tasks you do often. But that's for later — start with the basics!

## Tip

You can combine commands with regular messages:

```
You: /commit just added the login page
```
