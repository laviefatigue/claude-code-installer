# Claude in the Sidebar

You have two ways to use Claude:

## 1. Terminal (Claude Code CLI)

- Type `claude` in the terminal
- Great for: building projects, running commands, complex tasks
- Full power of Claude Code

## 2. Sidebar (Claude Extension)

- Click the Claude icon in the left sidebar
- Great for: quick questions, code explanations, small edits
- Stays visible while you work

## When to Use Which?

**Use the Terminal when:**
- You want Claude to create or modify files
- You're building something from scratch
- You need to run commands (tests, builds, etc.)

**Use the Sidebar when:**
- You have a quick question about code
- You want to chat while keeping your files visible
- You're learning and want explanations

## Working Together

You can use both at the same time!

- Keep the sidebar open for reference
- Use the terminal for the heavy lifting
- Copy code between them as needed

## Tip

Select some code in your editor, then ask about it in either place. Claude will see what you've highlighted.
