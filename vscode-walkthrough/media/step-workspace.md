## What You're Looking At

**Left Sidebar** — Icons for different views (files, search, source control, Claude)

**Editor** — Where you read and write code

**Terminal** — Where Claude Code lives (at the bottom)

**Status Bar** — Info about your current file (very bottom)

Don't worry about memorizing this — you'll learn it naturally as you work!
