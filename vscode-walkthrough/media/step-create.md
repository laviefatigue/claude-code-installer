# What Will You Create?

You're ready. The tools are set up. Claude is listening.

## Ideas to Try

### For Beginners
- "Create a personal webpage about me"
- "Make a simple game I can play in the browser"
- "Help me organize files in my Downloads folder"
- "Explain this code to me like I'm new to programming"

### For Makers
- "Build a habit tracker app"
- "Create a tool that resizes images"
- "Make a script that backs up my important folders"
- "Help me automate this repetitive task"

### For Learners
- "Teach me how loops work with examples"
- "Show me 5 different ways to solve this problem"
- "Review my code and suggest improvements"
- "What's the best way to learn [language]?"

## Remember

**You don't need to know how to code.**

Describe what you want in plain English. Claude will:
1. Figure out how to build it
2. Write the code
3. Explain what it's doing
4. Help you understand and modify it

## The Starting Line

This is where your creative journey begins.

Open the terminal, type `claude`, and tell it what you want to make.

What will you create?
