# The Terminal

The terminal is a text-based way to talk to your computer — and to Claude.

## Don't Be Intimidated

If you've never used a terminal before, that's okay! Think of it as a chat window where you type commands instead of clicking buttons.

## Quick Shortcuts

| Action | Shortcut |
|--------|----------|
| Open Terminal | `Ctrl + `` ` (backtick, below Esc) |
| New Terminal | `Ctrl + Shift + `` ` |
| Clear Terminal | Type `clear` and press Enter |
| Close Terminal | Click the trash icon or type `exit` |

## What You'll See

```
PS C:\Users\You\my-project>
```

This is called a "prompt" — it shows where you are (your folder) and waits for you to type something.

## Your First Command

Just type:
```
claude
```

And press Enter. That's it!
