Install Claude Code

AI coding assistant - step by step setup

1
Open PowerShell
Right-click the Start button and select "Windows PowerShell" or "Terminal"

What is PowerShell?

PowerShell is a command-line tool built into Windows. It lets you run commands by typing instead of clicking.

Think of it like texting your computer instead of pointing at things.

Tip: You can also press Win + X then click "Terminal" or "PowerShell"

2
Copy this command
Click the Copy button, then paste it into PowerShell
irm https://raw.githubusercontent.com/laviefatigue/claude-code-installer/master/install.ps1 | iex
To paste: right-click in PowerShell, or press Ctrl+V

What does this command do?

irm = "Invoke-RestMethod" - downloads a file from the internet

https://...install.ps1 = the installer script (view source)

| = "pipe" - sends the downloaded script to the next command

iex = "Invoke-Expression" - runs the script

3
Press Enter and wait
The installer will download and set up everything automatically. You'll see progress like this:
[1/6] Git for Windows (REQUIRED)
Already installed: 2.43.0

[2/6] Claude Code CLI (REQUIRED)
Installing...

What's happening?

The installer checks if each tool is already installed. If not, it downloads and installs it automatically.

Git - Required. Claude Code uses this to run terminal commands.

Claude Code - The AI assistant itself.

VS Code + Extensions - Code editor with Claude Code and Foam extensions pre-installed.

Node.js, Python - Runtimes so everything "just works" later.

4
Sign in to Anthropic
A browser window will open. Sign in with your Claude Pro, Max, or Teams account.
🔐
Browser opens automatically

Why sign in?

Claude Code connects to Anthropic's servers to provide AI assistance. Your account verifies you have access.

Don't have an account? You need a paid Claude subscription (Pro, Max, or Teams). Get one here

5
Start using Claude Code
Open any terminal and type:
claude
Or open VS Code, press Ctrl + ` to open the terminal, then type claude

You're ready!

Try these commands:

/help - See what Claude Code can do

/getting-started - Interactive tutorial

Or just type naturally: "Help me write a Python script that..."

1
Open Terminal
Press Cmd + Space, type "Terminal", and press Enter

What is Terminal?

Terminal is macOS's built-in command-line tool. It lets you run commands by typing instead of clicking.

Think of it like texting your computer instead of pointing at things.

Tip: You can also find Terminal in Applications > Utilities

2
Copy this command
Click the Copy button, then paste it into Terminal
curl -fsSL https://raw.githubusercontent.com/laviefatigue/claude-code-installer/master/install.sh | bash
To paste: press Cmd+V in Terminal

What does this command do?

curl -fsSL = downloads a file from the internet silently

https://...install.sh = the installer script (view source)

| = "pipe" - sends the downloaded script to the next command

bash = runs the script in the Bash shell

3
Enter your password if asked
Some installations require admin access. Type your Mac password when prompted (you won't see it as you type).

VS Code + Extensions - Code editor with Claude Code and Foam extensions.

Node.js, Python - Runtimes so everything "just works" later.
