The Journey Starts Here

Five quick stops and you're building with AI. We'll explain everything along the way.

1
First Stop
Open PowerShell
Right-click the Start button and select "Windows PowerShell" or "Terminal"
PS C:\Users\You>
?
What's PowerShell?

It's a command-line tool built into Windows. Think of it like texting your computer instead of clicking around.

Quick way: Press Win + X then click "Terminal"

2
Main Event
Run the Installer
Copy this command, paste it into PowerShell, and press Enter
irm https://raw.githubusercontent.com/laviefatigue/claude-code-installer/master/install.ps1 | iex
?
What does this do?

irm downloads our installer script

| sends it to the next command

iex runs the script

Want to read it first? View source on GitHub

3
Sit Back
Watch It Work
The installer handles everything. You'll see progress like this:
[1/6] Git for Windows
Already installed: 2.43.0

[2/6] Claude Code CLI
Installing...
?
What's being installed?

Git - Required for Claude Code's terminal

Claude Code - The AI assistant itself

VS Code + Extensions - Your code editor, ready to go

Node.js & Python - So everything "just works" later

4
Quick Detour
Sign In
A browser window opens. Sign in with your Claude account.
🔐
Browser opens automatically
?
Why sign in?

Claude Code connects to Anthropic's AI servers. Your account gives you access.

Need an account? You'll need a Claude Pro, Max, or Teams subscription.

5
Destination
Start Building
Open any terminal and type:
claude

Or in VS Code: press Ctrl + ` then type claude

1
First Stop
Open Terminal
Press Cmd + Space, type "Terminal", press Enter
you@mac ~ %
?
What's Terminal?

It's macOS's built-in command-line tool. Think of it like texting your computer instead of clicking around.

Alternative: Find it in Applications > Utilities

2
Main Event
Run the Installer
Copy this command, paste it into Terminal, and press Enter
curl -fsSL https://raw.githubusercontent.com/laviefatigue/claude-code-installer/master/install.sh | bash
?
What does this do?

curl -fsSL downloads our installer script

| sends it to the next command

bash runs the script
