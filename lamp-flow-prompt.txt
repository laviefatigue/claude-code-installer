TECHNICAL IMPLEMENTATION:

```javascript
// Capture scroll without moving page
let scrollProgress = 0;
const maxScroll = 5000; // Total "scroll distance" for full animation

window.addEventListener('wheel', (e) => {
  e.preventDefault();
  scrollProgress = Math.max(0, Math.min(maxScroll, scrollProgress + e.deltaY));
  const progress = scrollProgress / maxScroll; // 0 to 1
  updateAnimation(progress);
}, { passive: false });
```

CSS APPROACH FOR FLOWING COLORS:
```css
.color-flow {
  position: fixed;
  inset: 0;
  background: radial-gradient(
    ellipse at 50% 60%, /* origin at lamp spout */
    var(--current-color) 0%,
    var(--current-color) calc(var(--flow-radius) * 1%),
    transparent calc(var(--flow-radius) * 1% + 20%)
  );
}
```
