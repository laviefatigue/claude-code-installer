The Journey

We thought of everything so you don't have to.

1

Checks what you already have

The installer scans your system to see if you already have tools like Git, VS Code, or Node.js installed. It skips anything you already have, saving you time and bandwidth.
2

Downloads only what's missing

Efficient and respectful of your resources. We only fetch the components you need to complete the setup, ensuring a fast installation process without bloat.
3

Configures everything automatically

No more wrestling with PATH variables or config files. We handle all the wiring under the hood so all your tools talk to each other perfectly from the start.
4

Opens Claude Code ready to go

The moment installation finishes, Claude Code launches. You can start building immediately with your first command. No restarts required.

The installer is open source

Trust through openness. Read every line before you run it. We believe in complete transparency about what runs on your machine.
