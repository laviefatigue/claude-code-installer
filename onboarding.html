Step 3 of 6

Open a project folder

Claude works best when it can see your files.

Visual Studio Code

No folder opened

Click to open a folder

Or use: Ctrl+K Ctrl+O

📁 Why folders matter

When you open a folder, Claude can:

  • Read your files — understand your code
  • Create new files — in the right location
  • Edit existing code — make changes for you
  • Run commands — from your project directory

Tip: Create a new folder called my-first-project if you don't have one yet.

Two ways to add context:

1. Select code in the editor, then ask about it

2. Mention files by name: @index.html

Claude
Fix the bug in @index.html
I'll look at index.html and find the issue...
Type @ to reference files...

📎 Ways to add context

  • @filename — Type @ to reference specific files
  • Select code — Highlight code in the editor before asking
  • Paste errors — Copy error messages directly into the chat
  • Describe the goal — "I'm building a todo app" gives Claude context
