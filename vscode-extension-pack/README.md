# Claude Code Framework - Extension Pack

Everything you need to start creating with Claude Code.

## What's Included

### [Claude Code](https://marketplace.visualstudio.com/items?itemName=anthropic.claude-code) (Official)
The official Claude Code extension from Anthropic. Chat with Claude directly in VS Code, edit code, run commands, and build projects through natural conversation.

### [Foam](https://marketplace.visualstudio.com/items?itemName=foam.foam-vscode)
A personal knowledge management system that links your notes together. Great for documenting projects, learning, and building a connected notebook.

## Getting Started

After installing this extension pack:

1. **Open Claude** — Click the Claude icon in the left sidebar
2. **Sign in** — Connect your Anthropic account
3. **Start creating** — Type what you want to build
