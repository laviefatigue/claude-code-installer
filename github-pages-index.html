Claude Code Framework

A quiet framework for luminous work.

The Invitation

Claude Code Framework is a small lantern for builders: one installation that gathers the terminal, editor, extensions, runtimes, and starter rituals into a single place, so the next idea arrives to a workspace already listening.

What it does

Sets up the essentials around Claude Code so the first session feels composed instead of assembled.

How it feels

Less like provisioning a machine, more like opening a studio where the tools are already laid out.
