BOOK CSS APPROACH:
```css
.book {
  position: relative;
  transform-style: preserve-3d;
  perspective: 1500px;
}

.book-cover {
  position: absolute;
  width: 300px;
  height: 400px;
  background: linear-gradient(135deg, #8B4513, #654321);
  border-radius: 0 8px 8px 0;
  transform-origin: left center;
  transform: rotateY(calc(var(--open-angle) * 1deg));
  box-shadow: 2px 2px 20px rgba(0,0,0,0.4);
}

.book-pages {
  /* Stack of pages visible when book opens */
  background: linear-gradient(to right, #f5f0e1, #fff9e6);
}
```
